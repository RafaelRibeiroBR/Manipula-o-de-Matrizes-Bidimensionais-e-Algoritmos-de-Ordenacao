Descrição da classe Matriz

A classe Matriz representa uma matriz de inteiros, com atributos para armazenar os elementos e suas dimensões (linha e coluna).
Ela possui métodos para: preencher manual ou automaticamente, exibir a matriz e ordenar os elementos por linha, por coluna ou completamente.

Lógica utilizada

Preenchimento manual: percorre cada posição e pede um valor ao usuário.

Preenchimento automático: gera valores aleatórios usando Random.

Exibição: imprime a matriz linha por linha.

Ordenação por linha/coluna: aplica Bubble Sort em cada linha ou coluna.

Ordenação completa: achata a matriz em um vetor, aplica Merge Sort recursivo e reconstrua a matriz.

Comparação dos algoritmos

Bubble Sort: iterativo, simples, funciona bem para matriz pequena ou quase ordenada, mas lento para matriz grande (O(n²)). 

Merge Sort: recursivo, eficiente para matriz grande (O(n log n)), ordena todos os elementos de uma vez.

Minha principal dificuldade foi percorrer linha e coluna corretamente e implementar a ordenação completa.
No final, consegui criar uma matriz funcional, com diferentes tipos de ordenação, consolidando conceitos de programação e algoritmo.
